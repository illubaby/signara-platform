Test File Operations

Open files with their default applications (Excel, Word, PDF, etc.)

Open File Test

How to use:

1. Enter the full path to any file on your computer

2. Click "Open File" button

3. The file will open in its default application

Examples:

  • C:\Users\YourName\Documents\file.xlsx → Opens in Excel
  • C:\Users\YourName\Desktop\document.pdf → Opens in PDF viewer
  • /home/user/report.xlsx → Linux path (if on Linux)

Request Body:

}

JavaScript Usage:

await openFileWithDefaultApp('C:\\path\\to\\file.xlsx');
